# Discord Markdown Editor

このWebアプリはDiscordで使用できるMarkdown記法を使ったメッセージを編集して、その場でプレビューできるユーティリティです。

# 技術的情報

- React 19.2.0
- React-dom 19.2.0
- React-markdown 10.1.0
- Node.js 24.10.1
- TypeScript
- Vite

# 使い方

1. このプロジェクトの[GitHubページ](#)にアクセスする
2. メッセージを左側のフィールドに入力する
   ✅️ フィールドの上のボタンをクリックするとマークダウンを挿入できます
3. 入力したメッセージのプレビューを右側のエリアで確認する
4. 「COPY TO CLIPBOARD」を押して、入力したメッセージをコピーする

# How to Use

1. Access this project's [GitHub Pages](#)
2. Write your message in the left area
   ✅️ You can insert Markdown by pressing the button above the text input field
3. Check your message's style in the right area
4. Press "COPY TO CLIPBOARD" to copy your message to your clipboard
